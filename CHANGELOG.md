# Changelog

## 0.3.3

- Made the Problems tab easier to use.
- Moved **Copy problem details** where it is easier to see.
- Added a clear message when problem details are copied.
- Added a clear fallback message if Windows cannot copy to the clipboard.

## 0.3.2

- Added better problem reports when an app fails to start.
- Saved recent problem reports after LaunchBridge restarts.
- Fixed a table header in Installed Apps.

## 0.3.1

- Added the browser helper for Smart Click.
- Added support for simple package files that do not need a full manifest.
- Kept older package formats working.

## Older Releases

Older releases improved app launching, stopping, updates, browser tabs, and repair reports.
