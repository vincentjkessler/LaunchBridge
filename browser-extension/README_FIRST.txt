LaunchBridge Smart Click v0.2.0

After LaunchBridge updates this folder:
1. Open edge://extensions or chrome://extensions.
2. Press Reload on LaunchBridge Smart Click.
3. Click the extension toolbar icon to test the native connection.
4. Click a ZIP or LaunchBridge package link in ChatGPT, Gemini, or Claude.

Badge meanings:
1  = a download click was seen
…  = the download is being tracked
OK = the file was sent to LaunchBridge
!  = the native helper is not connected
ON = the connection test passed
