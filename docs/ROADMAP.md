# LaunchBridge Roadmap

## v0.2 foundation — implemented

- Extension Profiles.
- Immediate AI export and icon prompts.
- Manifest-free smart packages.
- Filename identity/version parsing.
- Smart launch-target detection.
- Custom extension icons.
- Existing advanced manifest compatibility.
- Turbo Launch, lifecycle supervision, Error Cockpit, and compile-before-replace updates.

## Next

1. Portable Extension Profile import/export UI.
2. Ambiguous launch-target chooser and remembered selection.
3. Browser companion extension with native messaging.
4. Standalone package validator CLI.
5. Publisher identities, signatures, trust policies, and revocation.
6. Sandboxed first launch and capability declarations.
7. Organization registries, private channels, and policy management.
8. Cross-platform protocol implementations.
